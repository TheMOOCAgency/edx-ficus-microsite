Foire aux questions (F.A.Q)

Comment s'inscrire à un MOOC ?

Pour s’inscrire à un cours, il faut d’abord créer un compte. Pour cela, cliquez sur le bouton « Créer un compte » de la page d’accueil de ce site [partie soulignée avec lien mooc-conservation.org]. Une fois inscrit, cliquez sur le cours qui vous intéresse et, sur la nouvelle page, cliquez sur « Je m’inscris ! ».

Comment créer un compte ?

Cliquez sur le bouton « Créer un compte » de la page d’accueil de ce site ou suivez ce lien [lien vers page register]. Remplissez le formulaire avec toutes les informations demandées, et cliquez sur le bouton « Créer mon compte mooc-conservation ». Un mail de confirmation vous sera envoyé à l’adresse mail renseignée dans le formulaire. Rendez-vous donc dans votre boîte de réception, ouvrez le mail et cliquez sur le lien qu’il contient. Ca y est, vous avez un compte, vous pouvez maintenant vous inscrire aux MOOC.

NB 1 : Il faut mettre le nom et prénom que vous souhaitez voir figurer sur vos attestations. Ces dernières seront générées automatiquement à partir des informations fournies dans ce formulaire.

NB 2 : Il faut utiliser une adresse mail existante, et ne pas en inventer une nouvelle.

Je n’ai pas reçu mon mail de confirmation. Que faire ?

Regardez dans vos spams/pourriels/courrier indésirable. Ce mail est envoyé automatiquement, et il est donc forcément envoyé à l’adresse renseignée dans le formulaire d’inscription. Vérifiez donc bien que ce mail ne se trouve pas dans votre mail, ou que vous avez entré la bonne adresse mail.

J’ai oublié mon mot de passe

Dans le formulaire de connexion, au-dessus du champs « mot de passe », cliquez sur « Mot de passe oublié ? ». Un nouveau formulaire s’affiche (pop-up), entrez votre adresse mail (la même utilisée lors de l’inscription), et suivez les instructions du mail qui vient de vous être envoyé. Si vous n’avez pas reçu le mail, vérifier vos spams/pourriels/courrier indésirable.

Mon mot de passe ne fonctionne pas

C’est que le mot de passe que vous avez entré est incorrect. Procédez comme pour l’oubli de mot de passe

J’ai déjà été inscrit lors de la session précédente, faut-il que je recrée un compte ?

Si vous avez participé à des MOOC avant janvier 2021, il faudra recréer un compte puisque nous avons changé de plateforme.

Pour toute inscription effectuée à partir janvier 2021, vous n’avez pas à recréer de compte à chaque session, il suffit de vous connecter avec votre compte existant, « Explorer les cours » et vous inscrire aux MOOC qui vous intéressent.

Comment faire pour recevoir mon attestation de réussite ?

Sur cette nouvelle plateforme, les attestations de réussite sont générées automatiquement à partir des informations fournies lors de votre inscription. Donc si vous avez obtenu une moyenne supérieure à 75% aux examens, vous pourrez télécharger votre attestation tout de suite après validation [paragraphe à détailler après finalisation de la plateforme].

Où ont lieu les examens ?

Tous les examens des MOOC se font en ligne.

Est-ce que je peux m’inscrire en cours de session ?

Oui, chacun s’organise à son rythme, et a jusqu’à la date butoir de chaque session pour passer les examens.

Les MOOC sont-ils gratuits ?

Oui.

Mon compte est bloqué après plusieurs tentatives de connexion infructueuses, que faire ?

Vous avez tenté de vous connecter trop souvent avec un mauvais mot de passe. Attendez que votre compte soit débloqué (24h ?) puis tentez de vous connecter avec le bon mot de passe. Si vous ne vous souvenez plus de votre mot de passe reportez-vous à la question 4 « J’ai oublié mon mot de passe ».
